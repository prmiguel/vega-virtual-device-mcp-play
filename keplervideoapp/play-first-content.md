# Play First Content Prompt

Please use the `vega-virtual-device` mcp server to automate the playback of the first content in the Kepler Video App by following these steps:

1.  First, launch the application using the ID `com.amazondeveloper.keplervideoapp.main`.
2.  Next, verify that the application has started and is currently running.
3.  Once the app is ready, press the **Down** button to navigate.
4.  Then, press the **Enter** button.
5.  Finally, press the **Enter** button again to select and play the content.
6.  Wait for 10 seconds to allow the content to load and play.
7.  Take a screenshot to capture the playback state.
8.  Terminate the application using the ID `com.amazondeveloper.keplervideoapp.main`.
